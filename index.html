About Me

Hello! My name is Alfredo. I've been learning web development for almost a year now and I wanted to create this website to be able to put into practice what I have learned in this time and also to be able to show some of the projects that I have been working on. The projects are small, but they have helped me understand a little more about how this tool that we all use every day called the Internet works.

This website has been a challenge for me, considering that I am mostly self-taught but I have learned a lot and I hope in the future to create tools that can help solve real-life problems.

If you want to know a little more about me, here is my resume and if you have any questions, do not hesitate to send me a message through the contact section. Thank you very much for visiting!
